Browse log files collected by a log_roller subscriber node.

Behaviours: gen_server.

Description

Browse log files collected by a log_roller subscriber node. log_roller_browser is a gen_server that is started by the log_roller application when the "-log_roller_type subscriber" argument is given in the shell.
