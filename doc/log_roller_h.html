An event handler that broadcasts log messages to subscribers.

Behaviours: gen_event.

Description

An event handler that broadcasts log messages to subscribers. log_roller_h is a gen_event that is registered with the error_logger and receives a copy of any progress, info and error reports. A list of subscriber nodes is maintained in the gen_event state and all log messages are broadcast to that list of nodes.
